```
[Имя персонажа] (игрок: [имя игрока])

[текст биографии]

Черты (очки: 18/18):
- [название черты] (уровень N) — [аспект 1], [аспект 2], ..., [аспект N]

Флаги:
- [текст флага] [тип, зафиксирован]

Запас кубов: [текущий]/7
Опыт: [доступно] доступно ([заработано] заработано, [потрачено] потрачено)

```
