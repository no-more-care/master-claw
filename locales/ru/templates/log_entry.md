# Формат записи в лог

## Запись после броска
```
## [Сцена / момент]
**Действие:** <персонаж> — <что пытался сделать>
**Бросок:** [кубы] → X успехов vs сложность Y → успех/провал
**Рассказчик:** Мастер / Игрок
**Состояния:** <назначенные или "нет">
**Резерв:** X → Y
**Последствия:** <что конкретно изменилось в мире>
```

## Запись события мира
```
## [Игровой момент]
**Событие:** <название>
**Описание:** <что произошло>
**Участники:** <персонажи, НПС>
**Последствия:** <что изменилось>
**Затронутые флаги:** <если применимо>
```
