# Формат стартового персонажа

```
## [Имя персонажа]
**Концепт:** <одна строка — почему этого персонажа интересно играть>

**Биография**
<2–4 предложения: кто он, откуда, что им движет>

**Черты**
- <Название черты> (уровень N) — <аспект 1>, <аспект 2>, ...<N аспектов>
- ...
(очков распределено: 18/18)

**Флаги**
- <текст флага> [тип: отношение/личность/цель/убеждение]
- ...

**Запас кубов:** 7/7
```
