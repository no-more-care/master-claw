# BlackBirdPie — Справочник механик

Краткая шпаргалка по правилам. Авторитетный источник: `skills/rules/SKILL.md`.

## Пул кубов

Каждый применимый элемент = **+1 куб** (НЕ +уровень черты!):
- Каждая подходящая **черта**: +1
- Каждый подходящий **аспект**: +1
- Подходящий **флаг**: +1
- **Резервные кубы**: игрок решает сколько добавить (0..текущий запас)

**Бросок:** Nd6, каждый куб с результатом 4-6 = 1 успех (hit).

## Сложность

| Сложность | Уровень |
|-----------|---------|
| 2 | Легко |
| 3 | Средне |
| 4 | Сложно |
| 5 | Очень сложно |
| 6 | Экстремально |
| 7+ | Почти невозможно |

## Права рассказчика

| Результат | Кто нарратит | Формула |
|-----------|-------------|---------|
| Успехи > Сложность | **Игрок** | "Да, и кроме того..." |
| Успехи = Сложность | **ГМ** | "Да, но..." |
| Успехи = Сложность - 1 | **Игрок** | "Нет, но..." |
| Успехи < Сложность - 1 | **ГМ** | "Нет, и вдобавок..." |

## Резервные кубы

- Старт: 7 | Максимум: 7
- **Успех:** потраченные кубы ТЕРЯЮТСЯ
- **Провал:** потраченные кубы ВОЗВРАЩАЮТСЯ + игрок получает +1 (до макс 7)
- **Помощь:** отдать 1 куб другому; при успехе — потерян, при провале — возвращён

## Персонаж

- **Черты:** 3-7 штук, уровни 2-6, сумма = **ровно 18**
- **Аспекты:** количество = уровень черты; добавляются в игре, потом фиксируются
- **Флаги:** минимум 3, обязательно ≥1 "отношение" (relationship); фиксируются после установки
- **Опыт:** ~1 очко / 30 минут игры; повышение N→N+1 = N очков; новая черта на уровне 2 = 3 очка

## Типичные ошибки

1. **Черта уровня 6 = +6 кубов** — НЕВЕРНО. Каждая черта = +1 куб.
2. **Резервный куб = +2** — НЕВЕРНО. Каждый резервный = +1.
3. **"Стрессовые кубы"** — нет такого термина. Только "резервные кубы".
4. **Бросок без подтверждения** — НЕВЕРНО. Всегда объявить сложность и ждать ОК.
5. **ГМ описывает действия за игрока** — НЕВЕРНО. Только игрок решает, что делает его персонаж.
